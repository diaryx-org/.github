## Diaryx

This is a GitHub organization for the Diaryx project. You can read about Diaryx at <https://diaryx.org>.

**Diaryx is closed-source**, but much of its logic has been refactored into open-source repositories:

- [Fig](https://github.com/adammharris/fig) (metadata parser)
- [Twig](https://github.com/adammharris/twig) (document parser, not yet integrated)
- [Colophon](https://github.com/adammharris/colophon) (plaintext workspace logic, not yet integrated)
- [Leaf](https://github.com/adammharris/leaf) (rich text editor, not yet integrated)

Right now, only [adammharris](https://github.com/adammharris) has access. If you would like to contribute or comment, you can contact him at adam@diaryx.org.
